# Does Music Listening Affect Our Body?

This repository supports an undergraduate honours thesis study examining whether **music listening produces measurable physiological changes**.

## Overview

Participants listen to short music clips while we record:

- **ECG** (single-lead, Lead II–like) for heart activity
- **Respiration** (chest belt transducer)
- **Facial sEMG** (smile/brow muscle activity)

The aim is to build a clean, reproducible dataset and analysis pipeline to test whether aspects of music (e.g., tempo/rhythm) can be inferred from synchronized physiological signals.

## Status

## Ethics and data

Human participant data are collected under Ontario Tech University REB approval.  
Any shared outputs will be **de-identified**. No identifiable participant information will be stored in this repository.
